# HedronBot

## Debugging
- **Start debugging:** `npm run debug`
- **Continue in debug mode:** `c`
- **Inspect variables:** `repl`
- **Show more surrounding lines:** `list(10)`

## Testing
- **Run tests:** `npm test`
